# Tier Placeholder Tile

**Create custom placeholder boxes for TierMaker!**

Tired of having to download and upload images just to label tiers?
This userscript lets you create **editable placeholder tiles** (text + color + shape) directly on the site and drag them into your tier list instantly.

## âš™ï¸ Installation

### 1. Install Tampermonkey

[Download Tampermonkey](https://www.tampermonkey.net/) for your browser.

### 2. Install Tier Placeholder Tile

Click the image below or [install from Greasy Fork](https://greasyfork.org/en/scripts/554827-tier-placeholder-tile).

[![Install on Greasy Fork](https://greasyfork.org/vite/assets/blacklogo96-CxYTSM_T.png)](https://greasyfork.org/en/scripts/554827-tier-placeholder-tile)

### 3. Head to TierMaker

Visit [https://tiermaker.com/](https://tiermaker.com/)
If the script is active, a floating window with this icon will appear in the top-right corner:

![GUI Icon](https://i.pinimg.com/736x/ae/d8/49/aed849cca00ffd2a756b107ae91075f9.jpg)

You can then:

* Type text in the box.
* Pick a color using the wheel.
* Click **Create Placeholder** to generate it.
* Drag the thumbnail into the tier grid or click **Create + Insert** to auto-upload.
